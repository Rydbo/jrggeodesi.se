Om oss
JRG Geodesi är ett lokalt mätföretag baserat i Hyltebruk, Hylte kommun, Halland med 20+ års erfarenhet från Sveriges största infrastrukturprojekt.

Erfarenhet — från Öresundsbron till Hylte kommun
Öresundsbron (Malmö Stad):
Fotogrammetrisk bearbetning och produktion av fullskalig 3D-baskarta från flygfoton. Projekteringsunderlag för svenska landfästet vid Lernacken.

Lantmäteriet:
Tidigare verksam som Lantmätare. Produktion av fullskaliga baskartor från flygfoton. Djup kännedom om krav vid fastighetsbildning.

Boliden Gruva:
Mätningstekniska uppdrag inom gruvindustrin.

Hylte Kommun:
Projekteringsstöd, utsättning, mätning och ajourhållning av primärkartor.

Scandia Consult (Ramboll) & Geospace AB:
Infrastruktur- och byggnadsprojekt. Komplett fotogrammetrisk inventering av nedlagda Kockumsvarvet. Baskarta för nya stadsdelen Varvsstaden i Malmö.

Tjänster
Drönarmätning, Utsättning, Lasermätning, Tomtgräns, Nybyggnadskarta
